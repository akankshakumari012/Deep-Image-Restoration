# Deep-Image-Restoration
RESTORE is an interactive web tool that improves image quality by applying restoration techniques and allows users to compare original and restored images.
RESTORE is a web-based image restoration tool that improves the quality of damaged or low-quality images using modern enhancement techniques. The system allows users to upload images and apply restoration features such as denoising, sharpening, color correction, and upscaling.

This project provides an interactive interface where users can compare original vs restored images in real time.

🚀 Features

Image Upload (Drag & Drop or File Selection)

Image Restoration Processing

Before vs After Comparison

Image Zoom and Rotation

Multiple Restoration Models

Adjustable Enhancement Controls

Color Correction (Brightness, Contrast, Saturation)

Image Upscaling

Export Restored Images (PNG, JPEG, WEBP, TIFF)

Download Restored Image
